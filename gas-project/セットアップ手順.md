# 医療面接AI評価システム - GASセットアップ手順

## 事前準備

### AWS Bedrock APIキーの取得

1. [AWS Bedrock コンソール](https://console.aws.amazon.com/bedrock/) にログイン
2. リージョンを **東京（ap-northeast-1）** に設定
3. 左メニュー「APIキー」→「長期APIキーを生成」
4. 有効期限を設定（推奨: 365日）→「生成」
5. 表示されたAPIキーをコピーして安全な場所に保存

**料金**: 従量課金（約10〜15円/回、月300〜450円目安）
**モデル**: Claude Sonnet 4.6（高精度な評価、教育的フィードバック）

## GASプロジェクトの作成

### Step 1: プロジェクト作成

1. [Google Apps Script](https://script.google.com/) にアクセス
2. 「新しいプロジェクト」をクリック
3. プロジェクト名を「医療面接AI評価システム」に変更

### Step 2: コードファイルの配置

#### Code.gs（メインコード）
- デフォルトの `Code.gs` を開く
- 中身をすべて削除
- `Code.gs` ファイルの内容をすべてコピー＆ペースト

#### BedrockApi.gs（AI API呼び出し）
- ファイルの横の「＋」→「スクリプト」を選択
- ファイル名を `BedrockApi` に変更（.gsは自動付加）
- `BedrockApi.gs` の内容をすべてコピー＆ペースト

#### EvaluationPrompt.gs（評価プロンプト）
- ファイルの横の「＋」→「スクリプト」を選択
- ファイル名を `EvaluationPrompt` に変更（.gsは自動付加）
- `EvaluationPrompt.gs` の内容をすべてコピー＆ペースト

#### その他のファイル
同様に以下のファイルを作成:
- `SelfEvaluation.gs` - 自己評価処理
- `SheetOperations.gs` - Google Sheets読み書き
- `MailService.gs` - Gmail送信
- `Index.html` - メイン画面（HTMLファイルとして作成）
- `Stylesheet.html` - CSS（HTMLファイルとして作成）
- `JavaScript.html` - クライアントサイドJS（HTMLファイルとして作成）

### Step 3: APIキーの設定

1. 左メニューの「⚙ プロジェクトの設定」をクリック
2. 下にスクロールして「スクリプト プロパティ」セクション
3. 「スクリプト プロパティを追加」をクリック
4. プロパティ名: `BEDROCK_API_KEY`
5. 値: 取得したBedrock APIキーをペースト
6. 「スクリプト プロパティを保存」をクリック

### Step 4: 動作テスト

1. エディタに戻る
2. 上部の関数選択ドロップダウンで `testApiKeySetup` を選択
3. 「▶ 実行」ボタンをクリック
4. 初回は権限の承認が求められるので「許可」
5. 「実行ログ」に「Bedrock API接続成功」と表示されればOK

### Step 5: ウェブアプリとしてデプロイ

1. 右上の「デプロイ」→「新しいデプロイ」
2. 種類の横の⚙→「ウェブアプリ」を選択
3. 設定:
   - 説明: 「Phase 0 テスト版」
   - 次のユーザーとして実行: **自分**
   - アクセスできるユーザー: **全員**（学生がアクセスするため）
4. 「デプロイ」をクリック
5. 表示されたURLをコピー → これが学生に配布するURLです

## 使い方

1. デプロイURLにアクセス
2. Plaudからエクスポートした文字起こしテキストをテキストエリアに貼り付け
3. 匿名化チェックリストで個人情報を確認・置き換え
4. 評価レベルを選択（Level 1 / 2 / 3）
5. 自己評価を入力
6. 「AI評価を実行」ボタンをクリック
7. 約15〜20秒で評価結果が表示される
8. 大画面モニターで教員と学生が一緒に確認
9. 「結果をメールで送る」で学生のGmailに送信

## トラブルシューティング

| 問題 | 対処法 |
|------|--------|
| 「BEDROCK_API_KEYが設定されていません」 | スクリプトプロパティを確認 |
| HTTP 429エラー | レート制限。1分待って再実行 |
| HTTP 403エラー | APIキーの有効期限切れ。Bedrockコンソールで新しいキーを生成 |
| 「権限が必要です」 | 初回実行時の承認画面で「許可」を選択 |
| 結果が表示されない | ブラウザのコンソールでエラーを確認 |
| レスポンスが遅い | 通常15〜20秒かかります。ローディング画面を待つ |
